# Текстовый редактор

Современный веб-текстовый редактор с поддержкой HTML и BB-тегов.

## Возможности

- **Три режима редактирования**: Визуальный, HTML, BBCode
- **Форматирование текста**: жирный, курсив, подчеркивание, зачеркивание
- **Работа с изображениями**: drag & drop, загрузка, изменение размера, позиционирование
- **Пресеты стилей**: готовые стили для текста и блоков
- **Экспорт**: HTML, BBCode, текст
- **История изменений**: undo/redo
- **Автосохранение**
- **Адаптивный дизайн**

## Использование

```html
<div id="editorContainer"></div>

<script>
const editor = new Editor({
    container: '#editorContainer',
    mode: 'visual'
});
</script>
